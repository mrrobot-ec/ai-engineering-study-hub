# *AI Engineering* — legal access and companion material

Chip Huyen's *AI Engineering: Building Applications with Foundation Models*
(O'Reilly, platform ISBN 9781098166298) is a commercial 534-page book. A random
GitHub PDF is not an authorized substitute.

## Full book

- O'Reilly book page: https://www.oreilly.com/library/view/ai-engineering/9781098166298/
- O'Reilly AI-engineering learning page and trial: https://www.oreilly.com/online-learning/ai-engineering.html
- Ask your employer, university, or public library whether it provides O'Reilly
  Learning access; otherwise buy the print/ebook or use the publisher trial.

## Official free companion material

- Author repository: https://github.com/chiphuyen/aie-book
- The repository contains the table of contents, author-written chapter summaries,
  study notes, case studies, prompt examples, an appendix, and further resources.
- A local clone belongs in `companion-code/aie-book/`. It is companion material,
  not the complete commercial text.

## Suggested reading order with this library

1. Read the companion summaries to map the book's subjects.
2. Use `ROADMAP.md` and the local papers/books to go deep on each subject.
3. Use legal O'Reilly or library access for the book's integrated narrative and
   production guidance.
4. Build and evaluate the portfolio systems described in the roadmap; passive
   collecting is not a substitute for implementation and error analysis.
